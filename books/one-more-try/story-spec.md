## Parent page notes

標準六條，另加本書特有三條：
1. **賽前提問的標準答案**：當他問「輸了會怎麼樣」，用 p9 的劇本回答——比賽結束、說 Good game、明天再玩。答案每次一致，焦慮才有地方放。
2. 賴皮當下**不拿書出來讀**（規則二），在賽前的中性時間預演 p5–p6 的腳本。
3. p9 是全書鑰匙：Owen 打出 home run 但球隊輸球——刻意讓「我表現好」和「我們贏了」是兩件事。讀到這頁可以停下來問：Owen today was the game fun?
